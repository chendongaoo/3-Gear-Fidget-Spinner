# 3-Gear-Fidget-Spinner
A fidget spinner that is composed of 4 gears (3 outer gears and one inner gear). This will test my ability to create gears and navigate through 3D modeling.

One of the first times I have tried to create gears in any shape or form for 3D modeling. Not going to lie at first creating them manually sucked a lot but I was able to learn a lot, and this is a pretty dope mini project to do.
